# soc-ip-Comma
In day-to-day SOC operations, handling large volumes of IP addresses can be repetitive and error-prone.

🚀 Built a SOC Automation Tool using Python

To solve this, I developed a lightweight desktop tool that:

🔹 Removes duplicate IPs
🔹 Converts IP lists into comma-separated format
🔹 Performs real-time threat intelligence checks
🔹 Integrates with AbuseIPDB and VirusTotal APIs
🔹 Supports both IP and hash reputation lookup

🛠️ Tech Stack: Python, Tkinter, API Integration

💡 This tool helps SOC analysts reduce manual effort and improve investigation speed.

🔐 Runs locally with no data storage, ensuring security and privacy.

Next steps: Enhancing with SIEM integration and advanced threat scoring.
